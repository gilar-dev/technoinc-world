# TechnoInc MC

This application is built based on creator experience in his 5 year-old Minecraft survival world since created first time in 2021.

Visit the site on: https://technoinc.world

## About the site

This application site contains pure articles where it came from the actual story that creator had, since his world keeps growing continously the story or article is also following along.

There are several categories you can see such as Civilizations, Characters, Organizations, and etc. You can also join along with us by creating your own article story on contribution page!

If you're interested to create your first article story, you can check this link out: https://technoinc.world/contribution

## About the contribution page

The contribution page is used to create and modify article by editing it via contribution edit page if you already have your own article created. This article building system is built using the same logic as many big company such as Wordpress and Notion. It uses Content Management System (CMS) to built stackable content within your article schema.

The system is very similar to Minecraft sandbox building system where you can add, move, and delete your content as you want. It provides you several content types where you can use it to create or build your article schema like Heading type, Table type, Paragraph type and more!
